Développement d’un fichier CSS pour l’impression au format PDF d’une page web

Pour ce sujet d’examen, vous devrez développer un fichier HTML et un fichier CSS spécifique pour l’impression au format PDF de la page web.

Nous souhaitons développer un “outil web” qui permet de créer des lettres permettant la saisie d’une personne et/ou organise pour obtenir des documents concernant des algorithmes de prises de décisions individuelles. Nous donnons dans message un exemple de mail reçu concernant une demande d’algorithme de sélection en master. Vous devez à partir de ce texte construire une page HTML permettant, là ou cela vous semble nécessaire, de paramètrer la lettre produite au format PDF (à partir du CSS qui sera associé).

Vous devez rendre sur votre site github :

  1. un répertoire de nom ControlFinal
  2. contenant un fichier main.html contenant la pas-encore lettre et un fichier mainPDF.css contenant les règles permettant la production de la lettre au format PDF.
  3. modifier le fichier index.html de votre site pour avoir dès le début de la page un accès au fichier main.html

Vous devrez tenir compte des éléments suivants:

  1. La page web contiendra des blocs bouton, textarea et autres de même genre.
  2. Le fichier HTML devra correspondre à la construction d’une lettre (date, lieu, objet, formule d’appel, contenu de la lettre, formule de politesse, signature, logo en-haut à gauche, …). Vous pouvez créer les balises dont vous avez besoin.
  3. Dans la lettre, il y aura des zones pour le nom de l’envoyeur et le destinataire. Il y aura au moins aussi un bouton si
  4. Le fichier CSS devra être conçu de manière à ce que ces éléments s’affichent de manière adéquate lors de l’impression au format PDF. Par exemple, les boutons devront être bien visibles et les textarea devront être correctement alignés et espacés. Voir les exemples à la fin.
  5. Le fichier CSS devra être optimisé pour l’impression, en utilisant des règles CSS spécifiques telles que ‘@media print’.
  6. Le fichier CSS devra être bien organisé et commenté pour faciliter la compréhension de votre travail par le correcteur.

Voici quelques critères qui pourront être pris en compte pour la notation:

  • Qualité du code CSS: utilisation des règles CSS de manière adéquate, respect des standards CSS, etc.
  • Adaptation du design à l’impression: utilisation de couleurs et de polices de caractères harmonieuses pour l’impression, utilisation de marges et de padding de manière adéquate, etc.
  • Fonctionnalités du fichier CSS: prise en compte de l’affichage des éléments bouton et textarea lors de l’impression, etc.
  • Organisation et commentaire du code: utilisation de commentaires pour expliquer le fonctionnement du code CSS, utilisation de noms de classes et d’identifiants de manière cohérente, etc.
